# Davis Industries Financial Analysis  

This repository contains a comprehensive financial analysis report and supporting model evaluating Davis Industries’ historical performance, projected financial outlook, cash flow generation, and collateral position. The analysis uses structured financial data (Task 1 Answer.xlsx) to assess profitability, sustainability, and return timing, providing insights for financial decision-making, credit evaluation, and strategic planning.  

## Highlights  
- **Revenue Growth:** Increased from $353.1M (FY2015) to $480.3M (FY2017), reflecting sustained expansion.  
- **Margin Improvement:** Gross margin rose from 46.8% to 49.5%; net margin improved from 7.9% to 8.5%.  
- **Cash Flow Strength:** Positive recurring free cash flow expected under base case projections.  
- **Collateral Support:** Accounts receivable, inventory, and PP&E provide tangible asset backing.  
- **Credit Profile:** Debt/EBITDA and interest coverage ratios indicate manageable leverage with sufficient cushion.  

## Summary Findings 

- **Historical Performance:** Consistent revenue and net income growth, with stable operating margins in the mid-teens.  
- **Return Profile:** Operating leverage allows incremental revenue to translate into higher profitability and cash generation.  
- **Base Case Outlook:** Continued revenue growth, stable margins, and disciplined capex support ongoing free cash flow.  
- **Return Timing:** Free cash flow realized progressively, supporting debt service and reinvestment flexibility.  
- **Collateral Analysis:** Strong asset base enhances creditworthiness and provides downside protection.  
- **Recommendations:** Maintain margin discipline, monitor capex efficiency, and update projections regularly to manage risks.
